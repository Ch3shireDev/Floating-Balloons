# Floating Diagrams
Webpage for floating diagram design system. Written completely in JS on client side, saved locally.

## User stories

User can...

- [ ] create a new bubble
- [ ] move a bubble
- [ ] write text into a bubble
- [ ] select many bubbles
- [ ] remove a bubble
- [ ] remove many bubbles
- [ ] connect two bubbles
- [ ] set a hierarchy between bubbles
